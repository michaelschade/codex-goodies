# Codex Meta

This skill matters when the problem is not just "write better words," but "choose the right Codex surface." It helps you improve Codex itself: hooks, subagents, skills, `AGENTS.md`, reusable prompts, and the boundary between prompt fixes and structural fixes.

## Why You Might Care

- Use it when a workflow keeps getting heavier prompts instead of a cleaner hook, skill, or agent design.
- Use it when you want to package a reusable Codex behavior so other machines or collaborators can benefit from it.
- Use it when you need to decide whether something belongs in a prompt, config, automation, programmatic Codex flow, or a shared repo surface.

If you already know you need the full operating rules, examples, and references, jump to [`SKILL.md`](SKILL.md).
