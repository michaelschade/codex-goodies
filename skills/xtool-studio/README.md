# xTool Studio

This skill matters when a laser or engraving task depends on the real material, not just what looks centered on screen. It turns xTool Studio setup into a measurement-driven workflow so imported artwork is sized, placed, and verified against the actual physical surface.

## Why You Might Care

- Use it when Codex needs to import artwork and place it on a specific board, panel, or other target surface without drifting onto the bed or empty background.
- Use it when "Scale to fit" or camera framing is not trustworthy enough and you want placement derived from measured geometry instead of guesswork.
- Use it when recipe changes, staged passes, or left/right placement instructions need to stay grounded in fresh visible proof.

If you need the exact operating rules, terminology, and verification discipline, read [`SKILL.md`](SKILL.md).
